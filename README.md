# Leave Management System

A ServiceNow-based application developed to automate employee leave requests and approvals.

## Features
- Leave Request Submission
- Manager Approval Workflow
- Leave Duration Calculation
- Client-Side Validation
- Dashboard and Reports
- Status Tracking

## Technologies Used
- ServiceNow App Studio
- Flow Designer
- Business Rules
- Client Scripts
- Dashboards
